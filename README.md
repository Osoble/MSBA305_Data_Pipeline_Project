# MSBA 305 Data Pipeline Project

## Project Overview

This project implements an end-to-end data pipeline using the Olist Brazilian e-commerce dataset, public holidays data, and GDP data. The pipeline includes data cleaning, transformation, relational database design, PostgreSQL storage using Supabase, and analytical SQL queries.

## Technologies Used

- Python
- Pandas
- SQLAlchemy
- PostgreSQL
- Supabase
- dbdiagram.io

## Key Components

- Data acquisition and preprocessing
- PostgreSQL database schema design
- ERD design
- Data insertion into PostgreSQL
- Analytical SQL queries
- AI-assisted debugging and query generation

## Database

The project uses PostgreSQL through Supabase. The database includes tables for customers, orders, products, sellers, payments, reviews, holidays, and GDP data.

## Main Analytical Queries

- Monthly order volume and revenue
- Top product categories by revenue
- Average review score by customer state
- Holiday vs non-holiday order comparison

## AI Usage

AI tools were used to support query generation, debugging, schema refinement, and documentation.
